# v1.a - Warm Editorial Theme

Alternative design option for the Mobile Design System.

## Design Direction
- **Style:** Soft UI Evolution with editorial warmth
- **Primary:** Teal (#0D9488) instead of Material Blue (#1976D2)
- **Background:** Warm off-white (#FAFAF8) instead of pure white
- **Typography:** Georgia serif for headings (editorial feel)
- **Shadows:** Softer, warmer shadows
- **Radius:** Slightly more rounded corners

## How to Use

Open any HTML file in browser. Links to non-v1a screens go to original v1 versions.
To apply v1.a theme to any v1 screen, add after styles.css:
```html
<link rel="stylesheet" href="v1a/styles-v1a.css">
```
